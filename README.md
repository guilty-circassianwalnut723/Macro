## 🛠️ Common Issues

If Macro does not open:

- Make sure you extracted the ZIP file
- Check that the file is not still inside a compressed folder
- Try running the app as administrator
- Restart your PC and open it again

If images do not load:

- Check that the file type is supported
- Make sure the file is not damaged
- Move the images to a simple folder path like `C:\Images\MacroTest`

If the app runs slowly:

- Close other heavy apps
- Use smaller image sets
- Reduce image size if the app gives that option
- Confirm your graphics driver is up to date
